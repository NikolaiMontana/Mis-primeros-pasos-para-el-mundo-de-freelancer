# File Organizer.

Descripción: Script en Python para automatizar la organización de carpetas (Escritorio/Descargas) clasificando archivos por su extensión. (.png .jpg .pdf .mp4 .mp3 etc)

Tecnologías: Python 3.x, Librerías os y shutil.

Instrucciones:

Configura la variable ruta con la carpeta que deseas limpiar.

Ejecuta el script.

Los archivos se moverán automáticamente a subcarpetas (Fotos, Documentos, etc.).

si la carpeta no existe creara una nueva.
